# 🔷🚀 Monorepo for @felixicaza/astro-capo and capo-rules

This monorepo contains the source code for both [`@felixicaza/astro-capo`](astro-capo) and [`capo-rules`](capo-rules) packages. Libs focused on providing a set of modern and accurate rules for `<head>` sorting in Astro applications and any other framework.

## 📦 Packages
- [`@felixicaza/astro-capo`](astro-capo): This package integrates the functionality of `capo-rules` for Astro applications.
- [`capo-rules`](capo-rules): This package contains the core set of modern and accurate rules for `<head>` sorting, it is a library designed for easy integration into any framework.
